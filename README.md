# EviChain — Inovação Probatória com Blockchain e IA

**EviChain** é uma plataforma de inteligência investigativa e denúncias voltada para conselhos de classe e órgãos fiscalizadores. Integra portal de captação, assistente de qualificação via IA e módulo de investigação digital com registro de evidências em blockchain.

## Funcionalidades

- **Portal de Denúncias** — Formulário inteligente com suporte a anonimato e categorização automática
- **Qualificação por IA** — Engine OpenAI para análise, verificação de elementos essenciais e enriquecimento de dados
- **Registro em Blockchain** — Hash SHA-256 com Proof of Work para imutabilidade e cadeia de custódia digital
- **Investigador Digital** — Robô que varre bases oficiais (CONFEF, CFM, OAB) para cruzar dados e gerar dossiês
- **Dashboard Analytics** — Métricas em tempo real, visualização de cadeia de custódia e relatórios exportáveis
- **PWA Multiplataforma** — Interface responsiva e progressiva com acesso offline
